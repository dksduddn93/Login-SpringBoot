# Login Spring Boot


### :blue_heart: 개요 :blue_heart:

> 아래의 5가지 인증 방식을 사용해 스터디
1. 세션과 쿠키를 이용한 인증
2. Access Token을 이용한 인증 (JWT)
3. Access Token + Refresh Token을 이용한 인증
4. OAuth 2.0을 이용한 인증
5. SNS 로그인
